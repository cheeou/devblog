Vue.js 세팅

Dec 16, 2021 • 1 min read

vuejs

1. About Vue.js

  • vue project를 생성하기 위해 npm을 이용하여 @vue/cli를 설치한다.
  • vue-cli는 vue project를 신속하게 생성해 주는 라이브러리이다.

  • npm은 node.js 기반으로 만들어진 각종 웹 개발 라이브러리 설치 도우미(pakage manager)이다. npm을 이용하려면 node.js 설치는 필수다.

  • 웹브라우저는 vue 소스코드를 해석할 수 없다.
    App.vue에 있던 파일을 index.html로 변환하여 실제 동작을 할 수 있도록 한다.

2. folder별 분류

  • node_modules: 프로젝트에 쓰는 라이브러리들 모음
  • src: 소스코드를 담는 폴더
  • public: html 파일, 기타 파일 보관
  • pakage.json: 라이브러리 버전, 프로젝트 설정 기록

3. vue 문법

✅ : 두 중괄호 안에 data 변수 명으로 가변적인 데이터를 연결해서, 실시간 자동 랜더링(data를 변경하면 변경 사항이 자동으로 html에 실시간에 반영됨)을 할 수 있다.
❶ html 속성에 바인딩은[ ':' ]로 바인딩 표시(ex :id="data 변수명")
❷ tag 사이 내용에 데이터 바인딩은

✅ v-for 반복 문법

<tag v-for="작명 in 반복횟수" :key="작명">

❸ :key는 반복문쓸 때 증가하는 변수로써(index 개념) 반드시 정의 해주어야함.
반복문 돌린 요소를 컴퓨터가 구분하기 위해 사용한다.
index개념으로 1씩 증가하여 반복되는 데이터의 변화값을 나타낸다.
